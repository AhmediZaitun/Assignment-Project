Q. 3 - Describe the various HTML Elements required to embed the video in an document and further explain the following HTML attributes :

Ans-The <video> HTML element embeds a media player which supports video playback into the document. We can use <video> for audio content as well, but the <audio> element may provide a more appropriate user experience.

  • <video> -The video tag is used to embed video content in a document, such as a movie clip or other video streams.
  • controls- it specifies that audio/video controls should be displayed. Controls should include: Play. Pause.
  • src-The src attribute specifies the location (URL) of the external resource.
  • poster-The poster attribute specifies an image to be shown while the video is downloading, or until the user hits the play button. If this is not included, the first frame of the video will be used instead.
  • width-The width attribute specifies the width of a video player, in pixels.If these attributes are set, the required space for the video is reserved when the page is loaded.
  • height-The height attribute specifies the height of a video player, in pixels.

<video width="320" height="240" controls source src="#" type="video/mp4"> <video>

Q. 4 - What do you mean by HTML Attributes? Explain the importance of HTML Attributes in terms of functionalities. Write a HTML document code explaining the various HTML attributes.

Ans-HTML attributes are special words used inside the opening tag to control the element's behaviour. HTML attributes are a modifier of an HTML element type.

  • The HTML title attribute provides additional "tool-tip" information
  • The HTML href attribute provides address information for links
  • The HTML width and height attributes provide size information for images
  • The HTML alt attribute provides text for screen readers
